# AI Career Navigator & Resume Analyzer

An AI-powered career guidance platform built using Python, Streamlit, Scikit-Learn, SQLite, Pandas, and Plotly.

## Features

- Resume Upload
- PDF Parsing
- Skill Extraction
- ML-Based Role Prediction
- Resume Score
- Skill Gap Analysis
- Learning Roadmap
- PDF Report Export
- Analysis History
- Dashboard Charts

## Technologies

- Python
- Streamlit
- Scikit-Learn
- SQLite
- Pandas
- Plotly
- ReportLab

## Run Project

pip install -r requirements.txt

streamlit run app.py
